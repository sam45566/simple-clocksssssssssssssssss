 Digital Clock App
A modern, minimalist digital clock web application that features a futuristic design and real-time synchronization with your system time.  

##Features
Real-Time Tracking: Automatically updates every second to display the current time.  
Responsive Layout: Uses CSS Flexbox to ensure the clock is centered on the screen.  
Vibrant Color Palette: Features high-contrast neon text against a dark charcoal background for maximum readability.  

###Setup Instructions
Create the files: Save the provided HTML, CSS, and JavaScript into files named index.html, style.css, and app.js.  
Organize your folder: Ensure all three files are located in the same directory for the links to function correctly.  
Launch: Open index.html in any modern web browser to view your clock.
